# Database Exploration & EDA Analytics Module

This isolated module provides dynamic schema reflection, database profiling, data quality auditing, and recommendation signals discovery for the **Talentree AI Platform**. It connects seamlessly to **MonsterASP SQL Server**, **Azure SQL**, **local SQL Server**, or any SQLAlchemy-supported SQL Server connection, and automatically falls back gracefully to JSON dataset files if the database is offline or empty.

## 🚀 Running Exploration Scripts

You can execute each script from the project root directory.

### 1. List All Database Tables
Discovers all tables in the SQL Server (or lists local fallback datasets).
```bash
python database_exploration/scripts/list_all_tables.py
```
*Outputs JSON report to:* `database_exploration/reports/all_tables_report.json`

### 2. Inspect Column Schemas
Profiles column types, nullability, defaults, PK and FK references for tables.
```bash
# Inspect specific table
python database_exploration/scripts/inspect_table_schema.py --table interactions

# Inspect all tables
python database_exploration/scripts/inspect_table_schema.py
```
*Outputs JSON report to:* `database_exploration/reports/schema_<table>.json`

### 3. Analyze Data Quality and Statistics
Computes row counts, duplicates, missing value percentages, numeric ranges, and datetime boundaries.
```bash
python database_exploration/scripts/analyze_table_statistics.py
```
*Outputs JSON report to:* `database_exploration/reports/statistics_<table>.json`

### 4. Detect Entity Relationships
Maps constraints, maps primary/foreign keys, and charts inferred recommendation joins.
```bash
python database_exploration/scripts/detect_relationships.py
```
*Outputs JSON report to:* `database_exploration/reports/relationships_report.json`

### 5. Export Slices of Data
Dumps a custom sample size of rows from any table for visual checks and debugging.
```bash
python database_exploration/scripts/export_table_samples.py --table products --rows 50
```
*Outputs JSON sample to:* `database_exploration/exports/products_sample.json`

### 6. Profiling Recommendation Signals
Automatically profiles behavioral signals for B2C Customers (purchases, clicks, views, repeats) and B2B Owners (procurements, reorders, seasonal trends, sparsity indices).
```bash
python database_exploration/scripts/analyze_recommendation_signals.py
```
*Outputs JSON report to:* `database_exploration/reports/recommendation_signals_report.json`

## 📊 Generating Premium Visualizations

To generate premium Seaborn visualizations of user event rates, popular products, categories, reorder timelines, temporal densities, and missing value heatmaps:

1. Install visualization libraries:
   ```bash
   pip install matplotlib seaborn
   ```
2. Execute the visualization generator:
   ```bash
   python database_exploration/visualizations/generate_eda_plots.py
   ```
3. Look at generated files in: `database_exploration/visualizations/`
   - `interaction_frequencies.png`
   - `popular_products.png`
   - `category_distributions.png`
   - `reorder_trends.png`
   - `temporal_activity.png`
   - `missing_value_heatmap.png`

## 📓 Running Interactive EDA Notebooks

To run interactive analytics notebooks in your local browser or editor:

1. Install Jupyter environment:
   ```bash
   pip install jupyter
   ```
2. Launch notebooks:
   ```bash
   jupyter notebook database_exploration/notebooks/
   ```
3. Navigate into the relevant subfolders (`database_exploration`, `customer_recommender`, or `owner_recommender`) to open the organized notebooks, inspect graphs, and run queries.
